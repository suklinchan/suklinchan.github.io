About this Visualization

This visualization project was created by D3.js as the final project for CS 498 Data Visualization, UIUC

Hybrid Structure

The visualization uses the interactive slideshow structure. At the top of the page, you can navigate through the project by clicking the next page buttons. Reader can select different views to look at data graph presented in each page.

Scenes

The narrative visualization is an interactive slide show with three scenes. The first scene is a bar plot which provides the number of drivers involved in fatal collisions by state. The second scene is a scatterplot of the number of drivers involved in fatal collisions vs. collisions due to any of 4 negligent driving behaviors observed for each state. The last scene is a scatterplot of the car insurance premium vs. collisions due to any of 4 negligent driving behaviors observed for each state.

User Interface Events

Mouse hoovering - a point on scatter plot and bars in the barchart will display the countryʻs name and the tooltips of the chart.

Mouse Clicking - navigates through the project pages and hyperlinks to data sources.

Annotations

Tooltips are used as annotation in both scatter plot and bar charts.When user's mouse move to selected bar, the points or bar along the X and Y axis in the charts provides additional annotations for the chart.
